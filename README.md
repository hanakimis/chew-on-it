# chew-on-it

A static collection of standalone, bite-sized learning courses:

- React fundamentals for reviewing agent-written code
- Practical typography through interactive specimens and tradeoffs

## Open

Open `index.html` directly, serve the folder locally, or publish it with GitHub Pages. Course progress is stored locally in the browser.

## Courses

- `index.html` is the React fundamentals course.
- `typography.html` is the practical typography course.

## Structure

- `index.html` is the default course and site entry point.
- `typography.html` is the typography course.
- `courses.json` tracks available courses and backlog ideas.
- `PRODUCT.md` captures product direction and design principles.

## Add A Course

1. Create a standalone course page, for example `testing.html`.
2. Add the course to the course switch navigation in each page.
3. Add the course to `courses.json`.

Keep courses self-contained when possible so they can be opened offline and copied independently.
